## Conventions

- One row per atomic mention. Multi-paragraph features are still one row.
- `URL` must point to a stable, citable location (publication URL · podcast RSS GUID · file on disk).
- When a mention is updated (e.g., a correction adds the `™` symbol), append a new row rather than editing the original.
- Self-published content (blog posts, social posts) is logged in `TRADEMARK_USAGE.md` and `FIRST_USE_LOG.md`, not here.
- This file is **not** the public-relations plan; it is the *evidence ledger* of what has actually been published.
